## [python/NLP] 감정분류(한국어)- 리뷰데이터 학습, 평가, 예측까지
티스토리 블로그에 포스팅한 감정분석을

1. preprocess_nlp.py : 전처리
2. train_nlp.py : 학습
3. test_nlp.py : 테스트

4개의 파일로 나누어 놓았습니다.
